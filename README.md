# Furniture Management System

Консольное приложение для управления каталогом мебели с использованием Spring Boot и PostgreSQL.

## Требования

- Java 17 или выше
- PostgreSQL 12 или выше
- Maven

## Настройка базы данных

1. Установите и запустите PostgreSQL
2. Создайте базу данных:
```sql
CREATE DATABASE furniture_db;
```

## Настройка приложения

1. Откройте файл `src/main/resources/application.properties`
2. Настройте подключение к базе данных:
```properties
spring.datasource.url=jdbc:postgresql://localhost:5432/furniture_db
spring.datasource.username=postgres
spring.datasource.password=postgres
```

## Запуск приложения

1. Склонируйте репозиторий
2. Перейдите в папку проекта
3. Выполните команду:
```bash
mvn spring-boot:run
```

## Использование

После запуска приложения вам будет доступно следующее меню:

```
Меню 
1. Добавить мебель
2. Показать все записи
3. Обновить запись
4. Удалить запись
5. Поиск по цене (дороже введённой)
6. Поиск по типу
0. Выход
```

### Примеры использования

#### 1. Добавление мебели
```
Выберите действие: 1
Введите название: Офисный стул
Введите тип: chair
Введите цену: 5000
```

#### 2. Просмотр всех записей
```
Выберите действие: 2
Furniture{id=1, name='Офисный стул', type='chair', price=5000.0}
```

#### 3. Обновление записи
```
Выберите действие: 3
Введите ID: 1
Новое название: Кресло офисное
Новый тип: chair
Новая цена: 6000
```

#### 4. Удаление записи
```
Выберите действие: 4
Введите ID: 1
```

#### 5. Поиск по цене
```
Выберите действие: 5
Введите минимальную цену: 5000
// Покажет всю мебель дороже 5000
```

#### 6. Поиск по типу
```
Выберите действие: 6
Введите тип: chair
// Покажет всю мебель типа 'chair'
```

## Структура проекта

- `model/Furniture.java` - Модель данных мебели
- `repository/FurnitureRepository.java` - Репозиторий для работы с БД
- `service/FurnitureService.java` - Бизнес-логика
- `console/ConsoleApp.java` - Консольный интерфейс
- `Application.java` - Главный класс приложения

## Зависимости

- Spring Boot
- Spring Data JPA
- PostgreSQL Driver

## Дополнительно

При возникновении проблем с кодировкой при выводе русского текста, выполните в базе данных:
```sql
ALTER DATABASE furniture_db SET client_encoding TO 'UTF8';
```
